# 楚创通 - 大学生全周期生涯规划多模态AI平台

## 项目简介

楚创通是专为全国高校与在校大学生打造的全周期生涯规划多模态AI赋能平台，聚焦大学生从入学适应、能力成长、方向定位到毕业发展的全链路成长需求。

### 核心理念

楚创通团队秉持 **"以生为基、以智赋能、以才兴邦"** 的核心理念，致力于通过AI技术打破教育资源壁垒与信息孤岛，为大学生提供个性化、系统化的全周期生涯发展支持。

### 团队Slogan

**AI赋能生涯路，共育时代栋梁才**

## 核心特性

### 三大自研AI引擎

1. **升学创 · 进阶引擎** - 覆盖专升本/考研/保研/留学全赛道，智能匹配，精准规划
2. **研赛创 · 循环引擎** - 科研成果转化，助力升学背景提升
3. **职途创 · 护航引擎** - 从升学背景到就业竞争力，无缝衔接

### 四大核心服务

- **升学规划** - 择校智能匹配、分赛道备考辅导、背景提升、面试全真模拟
- **学科竞赛** - 商业计划书撰写、路演PPT优化、竞赛项目评审与模拟答辩
- **科研辅助** - 论文全流程AI辅助、专利申报、科研项目评审优化
- **就业创业** - 职业测评、智能岗位匹配、求职材料优化、创业孵化

## 快速开始

1. 克隆或下载项目
2. 使用浏览器打开 `index.html` 即可预览
3. 或使用本地服务器：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js (需要http-server)
npx http-server -p 8000
```
