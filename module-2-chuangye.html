🎯 项目立项与市场调研

提交创业idea与核心成果，明确创业方向。AI进行行业赛道分析、市场规模测算、竞争格局拆解，生成目标用户画像；进行竞品对标分析、差异化亮点挖掘与SWOT初步分析。人工确认赛道定位与差异化竞争优势。

AI：行业赛道分析 + 市场规模测算 + 竞争格局拆解 AI：竞品对标分析 + 差异化亮点挖掘 + SWOT分析 人工：提交创意/核心成果 + 确认赛道定位
📊
02

💡 商业模式与项目框架构建

AI生成标准化创业项目大纲与商业模式画布设计，进行盈利测算、成本结构拆解与用户增长路径规划。人工调整盈利逻辑，确认核心业务流程与项目整体框架。

AI：生成项目大纲 + 商业模式画布设计 AI：盈利测算 + 成本结构拆解 + 增长路径规划 人工：调整盈利逻辑 + 确认商业模式与框架
🧩
03

📝 商业计划书编写与优化

AI分章节生成商业计划书初稿（执行计划、财务预测、市场分析、团队介绍、风险管控），进行语言逻辑优化、专业表述规范与数据图表可视化。人工进行核心内容强化、核心数据补充与专业审核。AI进行全文交叉校对、数据一致性校验与格式规范化优化，生成一页纸项目概要。

AI：分章节计划书初稿生成 AI：语言优化 + 图表可视化 + 全文校对 人工：核心内容强化 + 专业审核 + 最终确认
📄
04

💰 财务测算与风险管控方案设计

AI进行未来3-5年营收/成本/利润预测，盈亏平衡分析与现金流测算。人工调整财务假设，确认核心财务指标。AI识别项目风险点，设计风险管控方案与退出机制规划。人工确认风控方案与财务模型。

AI：财务预测 + 现金流测算 + 盈亏平衡分析 AI：风险识别 + 管控方案设计 + 退出机制规划 人工：确认财务模型 + 风控方案 + 最终审核
